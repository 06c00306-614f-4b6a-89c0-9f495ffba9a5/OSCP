# OS-CP+ Exam Report Requirements

## Submission Format
- **File**: `.7z` archive containing your PDF report
- **Time**: Submit within **24 hours** after exam ends
- **Naming**: `OSCP-OS-XXXXX-Exam-Report.7z`
- **No password** on archive
- **Max size**: 200MB

## Report Structure
```
1. Introduction
   - Methodology, Tools, Executive Summary

2. Findings (per machine)
   - IP, Services, Vulnerabilities
   - Exploitation walkthrough (step-by-step)
   - Privilege escalation steps
   - Proof.txt / local.txt screenshots

3. Active Directory Attack Chain
   - Enumeration → Lateral movement → Domain compromise

4. Remediation Recommendations

5. Conclusion
```
