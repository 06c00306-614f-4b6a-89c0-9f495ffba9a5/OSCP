# Exam Day Checklist

## Before Exam
- [ ] Fully charged laptop/battery
- [ ] Stable internet connection (backup ready)
- [ ] Screen recording software ready
- [ ] Tool lists ready (enum scripts, shell payloads)
- [ ] Report template ready
- [ ] Organized directory structure per machine
- [ ] VPN configuration checked

## During Exam
- [ ] Take screenshots OFTEN (every step)
- [ ] Note timestamps
- [ ] AD set FIRST (40pts guaranteed path)
- [ ] If stuck on one machine, switch to another
- [ ] Rotate machines every few hours
- [ ] Keep checking time remaining

## After Exam
- [ ] Compile report immediately
- [ ] Double-check all screenshots included
- [ ] Verify proof.txt / local.txt contents
- [ ] Create .7z archive
- [ ] Upload within 24 hours
- [ ] Verify MD5 hash matches uploaded file
